# lab1

## How to use
---
1. [Install Go](https://go.dev/doc/install) on your pc. If it is already installed skip this step.
2. Clone repository
```
git clone https://github.com/Gopack-go-labs/lab1.git
```
3. Change directory
```
cd lab1
```
4. Run the program
```
go run .
```
5. Go to [http://localhost:8795/time](http://localhost:8795/time)
